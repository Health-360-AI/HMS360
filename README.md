# HMS360 – Open Source Hospital Management System

HMS360 HMS is a **modern, cloud-ready, open-source Hospital Management System (HMS)** designed for clinics, hospitals, and health networks in emerging markets, with a strong focus on:

- Multi-branch / multi-site deployments
- Arabic / RTL localisation
- Interoperability (FHIR, HL7, DICOM)
- Hybrid cloud + on-prem architecture

This repository contains the **open-core edition** of HMS360.  

##  Key Features (Open Core)

- Patient registration & demographics
- Outpatient visits & encounters
- Basic inpatient admission/transfer/discharge (ADT)
- Appointments & scheduling
- Basic billing (cash, simple invoices)
- Basic pharmacy workflow (prescriptions & dispensing)
- Basic laboratory ordering & results entry
- Role-based access control
- REST/JSON API façade for integration
- Docker-based development environment


##  Project Status

> **Early stage – not yet production-ready.**  
> We welcome early adopters and contributors, especially hospitals and implementers in emerging markets.
